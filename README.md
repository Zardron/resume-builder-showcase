# Resume Builder Showcase

This is a public showcase repository for **Resume Builder**. The production source code is kept private to protect the full implementation, business logic, security-sensitive architecture, and project-specific assets.

## Live Project

https://resumeiqhub.vercel.app/

## What This Project Demonstrates

- Guided resume editing
- Template gallery
- Live preview
- PDF export flow
- Applicant dashboard

## Tech Stack

- React
- Vite
- Redux Toolkit
- Express
- MongoDB
- PDF export
- Tailwind CSS

## Public Repository Scope

This repository intentionally contains only a high-level folder outline and safe placeholder files. It is meant for portfolio review, project discovery, and architecture overview only.

The private source repository contains the complete application code, integrations, environment handling, database logic, authentication flows, and deployment configuration.
